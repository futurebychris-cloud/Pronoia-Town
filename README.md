# ☀️ Pronoia Town

An experimental AI town where autonomous agents live, remember, collaborate, and build things together.

> *Pronoia is the belief that the universe is quietly conspiring in your favor.*

Pronoia Town explores what happens when AI agents are given:

- Persistent memories
- Individual personalities
- Long-term goals
- Shared environments
- The ability to collaborate with humans and each other

Instead of isolated chatbots, every agent exists inside a living world.

---

## Features

- 🧠 Long-term memory
- 🤝 Multi-agent collaboration
- 🌎 Persistent AI world
- 💬 Natural conversations
- ⚡ Tool use
- 📅 Scheduling & planning
- 📚 Shared knowledge
- 🔍 Semantic memory search

---

## Vision

We believe AI will evolve beyond single conversations.

Future AI systems will inhabit persistent environments where they remember, learn, cooperate, and continuously improve over months or years.

Pronoia Town is an experiment toward that future.
